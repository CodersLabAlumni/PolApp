          About

          Purpose

          PollApp was created as an exercise ground for a variety of skills regarding software development. That included practical application of particular coding tools but also conceptualization, planning and work management, version control coordination, testing, deployment, etc.

          Despite the deployment at the current stage, the project lives and its original aims remain valid. Therefore we invite interested developers to take a look at the source code, which can be found under github.com/CodersLabAlumni/PollApp.

          We strongly believe in constant learning and community based development. Whether you're an experienced developer who can share his or hers knowledge, beginning developer looking for inspiration, a user who can share thoughts and suggestions regarding existing and future issues or a person wishing to propose some kind of cooperation, go ahead and contact us.

          You can find contact details under authors section.


          Used technologies and tools

          Our main focus is to develop competences regarding Java along with supplementing frameworks. However even the best backend engine seems divorced from reality without consideration for user experience. Hence we believe it is important to stay on top of balance between frontend and backend.

          Having said that, in the development of PollApp a variety of tools and technologies have been used, including:

          • Java
          • Spring
          • Hibernate
          • JavaScript
          • jQuery
          • REST
          • HTML
          • Bootstrap
          • Heroku
          • MySQL
          • Postgres
          • Github
